# Features:

**Real-time exchange rates** Leverage a powerful API to display accurate and up-to-date conversion rates.<br/>
**Simple and intuitive interface** Convert currencies effortlessly with a clean and user-friendly design.<br/>
**Multiple currencies** Support a wide range of currencies, allowing users to convert between popular and less common currencies.<br/>
**Customizable conversion** Specify the amount you want to convert and see the results instantly.<br/>

# Getting Started:

**Clone the repository** Use git clone https://github.com/your-username/currency-converter.git to clone this project.<br/>
**Install dependencies** Run npm install in the project directory to install all required dependencies.<br/>
**Start the development server** Run npm run dev to launch the application at http://localhost:3000 in your browser.<br/>


# Usage

Open the application in your browser.<br/>
Enter the amount you want to convert in the input field.<br/>
Select the currency you want to convert from the dropdown menu.<br/>
Select the currency you want to convert to from the other dropdown menu.<br/>
The converted amount will be displayed automatically.<br/>
